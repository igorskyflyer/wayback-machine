CANVAS (<canvas>) je HTML tag koji se pojavio u HTML-u 5. CANVAS se koristiti za crtanje grafike pomoću JavaScript-a. Može se koristiti na primer za crtanje grafova, foto-kompozicija, ili praviti proste animacije.

Canvas je prvi put predstavljen u Apple-u na Mac OS X-u, a kasnije je implementiran na pretraživaču Safari 1.3. Takodje ovaj tag podržavaju i Firefox 1.5, Internet Explorer 6, i Opera 9.

Canvas stvara crteže čiju površinu popunjavamo sa jednim ili više rendiranim kontekstima. Renderovanje se zasniva na OpenGL-u.

Na ovom sajtu ćemo objasniti kako ugraditi CANVAS element u HTML stranicu. Takodje ćemo dati i primere koji bi trebalo da opišu neke ideje šta možete učiniti sa CANVAS elementom.

Za početak evo jednog jednostavnog primera, čisto da biste shvatili kako funkcioniše CANVAS element.

  
  <canvas id="myCanvas" width="250" height="100" style="border:1px solid #d3d3d3;">
   Vaš pretraživač ne podržava HTML5 canvas.
  </canvas>

 <script>
  var c = document.getElementById("myCanvas");
  var ctx = c.getContext("2d");
  ctx.font = "30px Arial";
  ctx.fillText("Pozdrav svima", 25, 60);
 </script>
Vaš pretraživač ne podržava HTML5 canvas.
